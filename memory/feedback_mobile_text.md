name: feedback_mobile_text
description: Game is mobile-first — text must be large enough, graphics must be clear and recognizable
type: feedback
---

Text must not be too small — this is a mobile game. Increase font sizes across all scenes.

Graphics need to clearly communicate what objects are (window, desk, lamp, etc.) — pixel art silhouettes are too ambiguous at small sizes.

**Why:** User plans to play on mobile devices where small text is unreadable and unclear shapes are frustrating.

**How to apply:** Use minimum 14px for body text, 18px+ for important labels, 24px+ for titles. Make scene objects visually distinct with clearer shapes, outlines, or labels. Test at mobile viewport sizes.
